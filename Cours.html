Cours GIT

Quest-ce que GIT?

Git est un outil de versionning, et il permet d'avoir une traçacbilité et un historique de tous les fichiers du projet.

Il permet également de travailler en équipe: Envoyer nos modifications à nos collègues et récuperer les modifications des collègues.

Dans la traçabilité des fichiers, GIT nous donne toujours 3 infos: Qui, Quand, Pourquoi?

Installer et configurer GIT

git config --global user-email 'n.hoarau75@gmail.com'
git config --global user-name 'nohdevam'

Les commandes de base

Initialaiser un projet:

git init

Voir le statut des fichiers

git status

Ajouter des fichiers

git add "nom du fichier"
